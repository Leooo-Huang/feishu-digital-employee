# 步骤 6：线框图（ASCII）

对关键页面绘制 ASCII 线框图，展示布局和组件位置。

**示例格式：**
```
┌─────────────────────────────────┐
│  Logo    [搜索...]     [头像]   │
├─────────────────────────────────┤
│                                 │
│  ┌──────┐ ┌──────┐ ┌──────┐   │
│  │卡片 1│ │卡片 2│ │卡片 3│   │
│  │      │ │      │ │      │   │
│  └──────┘ └──────┘ └──────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ 列表项 1            [>] │   │
│  ├─────────────────────────┤   │
│  │ 列表项 2            [>] │   │
│  └─────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│  [首页]  [发现]  [+]  [我的]   │
└─────────────────────────────────┘
```

**要求：**
- 只画关键页面（首页、核心功能页、重要交互页），不是所有页面都需要
- 标注组件名称和交互方式
- 移动端和桌面端如果布局差异大，各画一版
