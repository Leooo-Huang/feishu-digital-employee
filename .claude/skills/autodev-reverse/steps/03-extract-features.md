# 步骤 3：提取功能

从代码中还原产品功能清单。

**提取方法：**

1. **从路由/页面提取**
   - 前端路由文件 → 用户可访问的页面列表
   - 后端路由文件 → API 端点列表
   - 每个路由 = 一个功能点或子功能

2. **从组件提取**
   - 搜索页面级组件，理解每个页面的功能
   - 搜索表单组件 → 用户输入/创建操作
   - 搜索列表/表格组件 → 数据展示操作

3. **从数据模型提取**
   - Schema/Model 定义 → 核心业务实体
   - 实体间关系 → 业务逻辑

4. **从 Git 提交提取**
   - 按功能分组提交（从 commit message 中识别 feature/fix/refactor）
   - 识别功能的演进顺序

**产出：**
- 功能清单（每个功能：名称、描述、对应代码位置）
- 功能分类：核心功能 / 辅助功能 / 基础设施
- 功能完成度初判（完整 / 半成品 / 桩代码）
