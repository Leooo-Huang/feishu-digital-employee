# 步骤 5：阶段 3 — UI/UX 设计

**目标**：调用 `autodev-ui` 技能，从用户流程出发设计页面结构和交互。

## 执行

调用 `autodev-ui` 技能。

**衔接：** 该技能会自动读取 `*-ideation.md`（用户故事、核心交互）和 `*-design.md`（架构、组件）作为输入上下文。

- **输入**：`*-ideation.md` + `*-design.md`
- **产出**：`docs/plans/YYYY-MM-DD-{slug}-ui.md`

## 验证

产出文件必须包含"页面清单"章节。
