## 7a. 实现微调检查

实现过程中方案可能有微调（比如设计时说用方案 A，实际发现不行换了方案 B）。快速检查：

- 阶段 4.5 更新的文档内容是否仍然准确？
- 如果有微调 → 补充修正到对应文档
- 如果没有微调 → 输出"实现与设计一致，无需修正"

## 7b. 状态标记

**不可跳过**：即使改动很小，也要输出 `--- 阶段 7 完成 ---` 确认执行了。
