# 步骤 3：功能发现 — 创意综合

**这是最核心的步骤。** 把三个维度的调研结果交叉碰撞，产生功能创意。

思考框架：

```
用户痛点 × 市场空白 × 前沿研究 × 技术可行性 = 有价值的功能
```

对于每个发现的功能点，记录：
- **功能名称**：一句话描述
- **来源**：哪个调研维度启发了这个想法
- **用户价值**：解决什么痛点 / 满足什么需求
- **创新点**：和现有方案比有什么不同
- **技术可行性**：粗略判断能不能做（高/中/低），不需要详细架构

## 创意激发方法

### 1. 组合法
把两个不相关的发现组合起来。

例："用户想看别人的 beta" + "骨骼识别技术" → "AI 自动生成攀爬路线的手脚点位置"

### 2. 对比法
在用户视频上叠加对比。

例："用户想和朋友比较" + "视频处理" → "双人路线回放对比，一个屏幕看两个人爬同一条线"

### 3. 转化法
把研究论文变成功能。

例："悬吊训练提高指力" → "个性化指力训练计划，按论文方法设定时间/组数/频率"

### 4. 差评驱动法
竞品差评 → 改进功能。

例：竞品差评"记录线路太麻烦" → "拍照自动识别线路等级和位置"
